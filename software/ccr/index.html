The Constraint Contextual Rewriting Project Home Page

The Constraint Contextual Rewriting project is concerned with the effective integration of decision procedures within formula simplification.

Keywords: Artificial Intelligence, Automated Reasoning, Simplification, Rewriting, Contextual Rewriting, Conditional Rewriting, Ordered Rewriting, Decision Procedures, Logic Programming, Constraints, Verification System, RDL.
